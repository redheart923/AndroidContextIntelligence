Android Context Intelligence canonical project payload

Source of truth:
  The tracked repository directory project/

Managed top-level files:
  .gitignore
  INSTALLATION_MANIFEST.txt
  README.md
  requirements-lock.txt

Managed source directories:
  collectors/
  config/
  configs/
  graph/
  queries/
  scripts/
  storage/
  tests/
  workspace/

Installed metadata:
  .android-context-installation.json

The installed metadata records the source commit and SHA-256 digest of every
managed payload file. Use the repository setup command or
scripts/verify_project_install.py to detect added, removed, or modified managed
files.

Preserved upgrade paths:
  .venv/
  data/
  config/source_roots.toml
  configs/local.yaml

Runtime databases, reports, virtual environments, caches, backups, vendor
inputs, and decompiler output are not part of this canonical payload.

Canonical rebuild entry point:
  scripts/rebuild_all.sh

Current source modules include Java, Kotlin tag, AIDL/Binder, Java inheritance,
service registration, Permission, atomic publication, and Vendor collection
foundations. Presence in this payload is not a claim of complete semantic or
live-database coverage; verified coverage is reported by each graph build.
